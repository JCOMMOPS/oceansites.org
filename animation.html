In this example setting, a seismometer at an OceanSITES reference station monitors an active fault zone. Fiberoptic cable links the sensor to a surface buoy, which transmits the data in real time to scientists on land. In actual deployments, the buoy and mooring line can be equipped with many sensors recording multiple kinds of data. About every year, research vessels visit each OceanSITES station for maintenance.
